1. Download the custom translation model from the following link (download the complete folder as it is) - https://drive.google.com/drive/folders/1EmCM96pY9Uh-9925LaLB48BLZT3jGu13?usp=sharing
2. Download the folder and save it in project root directory, named as "he-en-model".
3. Then run the following commands in the terminal in sequence -
   a. python -m venv venv
   b. [Select the venv's python from vscode's command pallete. Then create a new terminal.]
   c.  pip install -r requirements.txt.
   d. python run_app.py
